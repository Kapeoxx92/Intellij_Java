Stwórz klasę Temperature (oznaczoną jako final), która reprezentuje temperaturę. Klasa jest niemodyfikowalna — nie ma setterów, operacje zwracają nowe obiekty.

Klasa powinna posiadać:

    Prywatne final pole double celsius
    Prywatny konstruktor (nie można tworzyć obiektu przez new Temperature(...) z zewnątrz)
    Prywatną metodę pomocniczą create(double celsius) — rzuca IllegalArgumentException("Temperature below absolute zero") gdy celsius < -273.15

Statyczne fabryki (każda waliduje temperaturę):

    static Temperature ofCelsius(double celsius)
    static Temperature ofKelvin(double kelvin) — rzuca gdy kelvin < 0; przelicza: celsius = kelvin - 273.15
    static Temperature ofFahrenheit(double fahrenheit) — przelicza: celsius = (fahrenheit - 32) * 5 / 9

Konwersje (obliczane na bieżąco):

    toCelsius() → zwraca przechowane pole celsius
    toKelvin() → celsius + 273.15
    toFahrenheit() → celsius * 9.0 / 5.0 + 32.0

Metody porównawcze:

    isColder(Temperature other) → this.celsius < other.celsius
    isWarmer(Temperature other) → this.celsius > other.celsius
    isSameTemperature(Temperature other) → Double.compare(this.celsius, other.celsius) == 0

Metoda average(Temperature other) → zwraca NOWY obiekt z uśrednioną temperaturą: new Temperature((this.celsius + other.celsius) / 2.0) (użyj prywatnego konstruktora bezpośrednio).

toString() → Temperature[celsius=<celsius>]

Format wejścia — komendy:

    CELSIUS <value> — t = Temperature.ofCelsius(value)
    KELVIN <value> — t = Temperature.ofKelvin(value)
    FAHRENHEIT <value> — t = Temperature.ofFahrenheit(value)
    TO_C — wypisz toCelsius()
    TO_K — wypisz toKelvin()
    TO_F — wypisz toFahrenheit()
    PRINT — wypisz toString()
    COLDER <celsius> — wypisz t.isColder(Temperature.ofCelsius(celsius))
    WARMER <celsius> — wypisz t.isWarmer(Temperature.ofCelsius(celsius))
    AVERAGE <celsius> — t = t.average(Temperature.ofCelsius(celsius)); wypisz t.toString()
    ERR_C <value> — próba ofCelsius; jeśli wyjątek, wypisz message
    ERR_K <value> — próba ofKelvin; jeśli wyjątek, wypisz message

    toString() → Password[hash=<hash>, strong=<isStrong()>]

Format wejścia — komendy:

    CREATE <token> — tworzy Password (jeden token bez spacji)
    MATCHES <token> — wypisuje true/false
    STRONG — wypisuje isStrong()
    STRENGTH — wypisuje getStrength()
    CHANGE <oldToken> <newToken> — changePassword (jeśli wyjątek, wypisz message)
